## 왜 특정 국가 설정을 피하고, 추천인 코드를 신중히 입력해야 할까요?

*   **고수수료 국가로 분류될 위험**: 바이낸스는 사용자의 IP 주소, 신원 정보(KYC)에 제출한 주소지 등을 종합적으로 판단하여 사용자를 특정 관할권(국가)에 할당합니다. 일부 국가의 사용자들에게는 더 높은 거래 수수료, 제한된 서비스(예: 선물 거래 불가), 또는 특정 입출금 방법 사용 불가 등의 제약이 따를 수 있습니다. 초보자가 등록 시 주의하지 않으면, 실제 거주지와 관계없이 불리한 조건이 적용되는 국가로 잘못 분류될 수 있습니다.
*   **추천인 코드 입력 오류로 인한 영구적 혜택 상실**: 추천인 코드는 계정 생성 시 **단 한 번만** 입력할 수 있는 기회입니다. 코드를 입력하지 않거나, 유효하지 않거나, 낮은 할인율을 제공하는 코드를 입력하면, 해당 계정은 **평생 동안** 더 높은 수준의 수수료 할인(예: 20%) 및 신규 회원 보너스(예: 100 USDT 상당의 쿠폰)를 받을 기회를 완전히 상실합니다. 나중에 변경하거나 추가할 방법이 전혀 없습니다.

## 계정 보안 설정: 해킹으로부터 자산을 지키는 최후의 보루

거래소 가입이 끝났다면, 반드시 다음과 같은 보안 설정을 완료하세요. 이 단계를 생략하는 것은 금고 문을 열어두고 외출하는 것과 같습니다.

*   **2단계 인증(2FA) 설정**: '보안' 설정에서 Google Authenticator(구글 OTP)나 Authy와 같은 앱을 이용한 2FA를 반드시 활성화하세요. SMS 인증보다 훨씬 안전합니다.
*   **반피싱 코드 설정**: 바이낸스에서 발송하는 모든 공식 이메일 하단에 표시될 고유 코드를 설정하세요. 이 코드가 없는 이메일은 가짜 피싱 메일일 가능성이极高습니다.
*   **출금 화이트리스트 설정**: 자주 사용하는 출금 주소만 등록해 두고, 새로운 주소로 출금 시에는 추가 이메일 확인을 요구하도록 설정하면 자금 도난 위험을 크게 낮출 수 있습니다.

## 자주 묻는 질문 (FAQ)

**Q: 등록 페이지에서 '지원되지 않는 국가'라고 나옵니다.怎么办?**
A: 먼저 VPN을 완전히 종료한 후 다시 시도해 보세요. 그래도 안 된다면, 인터넷 공유기와 스마트폰의 데이터 연결을 모두 재시작(끄고 켜기)하여 IP 주소를 갱신한 후 다시 시도하세요. 여전히 문제가 있다면, 바이낸스 공식 텔레그램 또는 고객 지원팀에 문의하세요.

**Q: 추천인 코드 `ABCD1234`를 입력하지 않고 가입했습니다. 나중에 추가할 수 있나요?**
A: **절대 불가능합니다.** 추천인 코드는 계정 생성 과정에서만 입력이 가능하며, 한 번 생성된 계정에는 어떤 방법으로도 코드를 추가하거나 변경할 수 없습니다. 이는 바이낸스의 불변의 정책입니다. 코드 입력 기회를 놓쳤다면, 새로운 이메일로 계정을 다시 생성하는 것 외에는 방법이 없습니다.

**Q: KYC를 위해 제출한 개인정보는 안전한가요?**
A: 바이낸스는 업계 최고 수준의 암호화 기술과 데이터 보호 체계를 갖추고 있습니다. 사용자의 KYC 정보는 엄격한 규제를 준수하기 위한 목적으로만 사용되며, 제3자와 공유되지 않습니다. 또한, 정보 저장 시 암호화되어 보관됩니다.
