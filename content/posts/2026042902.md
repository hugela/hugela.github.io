## 1. 맞춤형 워크스페이스 레이아웃 설정하기

Bitget PC 버전의 가장 큰 장점 중 하나는 드래그 앤 드롭으로 자유롭게 구성할 수 있는 모듈식 인터페이스입니다. 기본 화면으로는 충분하지 않을 수 있습니다.

*   **차트 중심 레이아웃**: 주로 기술적 분석에 의존하는 트레이더라면, 차트 영역을 화면의 70% 이상으로 확장하세요. 나머지 공간에 주문창과 포지션 현황을 배치하면 시장 변화에 빠르게 대응할 수 있습니다.
*   **멀티 타임프레임 모니터링**: 하나의 심볼(예: BTC/USDT)에 대해 15분 차트, 1시간 차트, 4시간 차트를 각각 별도의 창으로 띄워 놓습니다. 이를 통해 단기 변동성과 장기 추세를 동시에 파악할 수 있어, 더 정확한 매매 타이밍을 잡는 데 도움이 됩니다.
*   **주요 지표 고정**: RSI, MACD, 볼린저 밴드 등 자주 사용하는 기술 지표를 설정한 후, 해당 설정을 '템플릿'으로 저장하세요. 새로운 차트를 열 때마다 매번 지표를 추가할 필요가 없어집니다.

> **실행 방법**: 화면 우상단의 '레이아웃' 아이콘을 클릭하여 '새 레이아웃'을 생성하거나, 기존 레이아웃을 자신의 트레이딩 스타일에 맞게 조정한 후 저장하세요. 이 설정은 브라우저에 저장되므로, 동일한 기기에서 접속할 때마다 완벽한 워크스페이스를 즉시 불러올 수 있습니다.

## 2. 조건부 주문(Trigger Order)을 활용한 정밀한 진입/청산

시장가나 지정가 주문만 사용한다면 중요한 기회를 놓치거나 불필요한 손실을 볼 수 있습니다. Bitget의 조건부 주문 기능은 가격이 특정 조건에 도달했을 때만 주문이 체결되도록 설정할 수 있어, 감정적 거래를 줄이고 전략을 철저히 수행하는 데 필수적입니다.

*   **돌파 매수(Breakout Entry)**: 지속적인 저항선을 관찰 중인 코인이 있다고 가정해 보세요. "트리거 가격"을 저항선 바로 위(예: $10,050)로 설정하고, "주문 가격"을 시장가 또는 약간 높은 지정가로 설정합니다. 가격이 저항선을 돌파할 때만 자동으로 매수 주문이 실행됩니다.
*   **손절매 스탑 로스(Stop-Loss)**: "트리거 가격"을 설정한 손절매 라인(예: $9,500)으로, "주문 가격"을 시장가로 설정합니다. 가격이 하락하여 해당 라인에 터치되면 즉시 청산 주문이 체결되어 추가 손실을 막아줍니다.
*   **이익 실현 트레일링 스탑(Take-Profit Trailing Stop)**: 일정 비율의 이익을 확보한 후, 추가 상승 여력을 놓치고 싶지 않다면 트레일링 스탑 기능을 사용하세요. 최고점에서 설정한 퍼센트(예: 5%)만큼 하락할 때까지 포지션을 유지하다가, 반전 시그널이 나타나면 자동으로 청산됩니다.

> **실행 방법**: 주문창에서 '조건부' 탭을 선택한 후, '트리거 가격', '주문 가격', '수량'을 입력합니다. 특히 변동성이 큰 시장에서는 반드시 이 기능을 활용하여 리스크를 관리하세요.

## 3. 고급 차트 도구와 온체인 데이터 연동 활용

Bitget PC 차트(TradingView 제공)는 강력한 분석 도구의 보물창고입니다. 기본 선과 도형 그리기 이상의 기능을 활용해 보세요.

*   **패턴 인식 도구**: '추세선 도구' 옆의 확장 메뉴에서 'Head & Shoulders'(두골 어깨형), 'Double Top'(더블 탑) 등의 자동 패턴 인식 도구를 찾을 수 있습니다. 이 도구는 차트에 잠재적인 반전 또는 지속 패턴을 표시해 줍니다.
*   **피보나치 확장/되돌림**: 단순한 피보나치 되돌림뿐만 아니라, '피보나치 확장' 도구를 사용하여 잠재적인 이익 실현 목표 구간을 설정할 수 있습니다. 이는 특히 파동 이론을 적용할 때 유용합니다.
*   **온체인 데이터 오버레이(선택 사항)**: Bitget은 일부 주요 온체인 지표(예: 거래소 유입/유출)를 제공합니다. 이러한 데이터를 차트 하단의 별도 창에 추가하여, 가격 움직임과 대형 지갑의 움직임을 비교 분석해 보세요. 불일치가 발생할 경우 시장 변곡점의 신호일 수 있습니다.

> **실행 방법**: 차트 상단의 도구 모음에서 다양한 아이콘을 탐색해 보세요. 각 도구 위에 마우스를 올리면 간단한 설명이 표시됩니다. 복잡한 도구는 공식 Bitget 아카데미의 관련 동영상 가이드를 참고하는 것을 추천합니다.

## 4. API 키를 통한 포트폴리오 관리 및 자동화 준비

PC 버전의 웹 인터페이스에서 직접 API(Application Programming Interface) 키를 생성하고 관리할 수 있습니다. 이는 수동 거래의 한계를 넘어서는 중요한 단계입니다.

*   **읽기 전용(Read-Only) 키 생성**: 먼저, 서드파티 포트폴리오 추적 앱(예: CoinMarketCap 포트폴리오, Delta)에 연결할 때는 '읽기 전용' 키만 생성하세요. 이 키는 자산 현황과 거래 내역을 조회할 수만 있을 뿐, 주문이나 출금은 절대 불가능하므로 안전합니다.
*   **트레이딩 봇/자동화 소프트웨어 준비**: 더 나아가 자동화 거래를 고려한다면, '거래(Trading)' 권한이 있는 API 키가 필요합니다. **절대 출금(Withdraw) 권한은 부여하지 마세요.** 이 키를 안전한 오프라인 환경에 보관하고, 신뢰할 수 있는 자동화 소프트웨어에만 연결하십시오.
*   **IP 화이트리스트 설정**: API 키 생성 시, 특정 IP 주소만 접근을 허용하는 'IP 화이트리스트' 기능을 꼭 활용하세요. 이는 키가 유출되더라도 지정된 IP 이외에서는 사용을 완전히 차단하는 강력한 보안층을 추가합니다.

> **실행 방법**: PC 버전 우상단의 프로필 아이콘 → 'API 관리'로 이동합니다. 키 생성 시 필요한 권한만 정확히 선택하고, 비밀 API 키(Secret Key)는 생성 즉시 안전하게 보관하세요. 화면에 다시 표시되지 않습니다.

## 자주 묻는 질문 (FAQ)

**Q: PC 버전과 모바일 앱의 주문이나 설정이 실시간으로 동기화되나요?**
A: 네, 완벽하게 동기화됩니다. PC에서 설정한 조건부 주문이나 레이아웃은 동일한 계정으로 모바일 앱에 로그인해도 확인 및 관리할 수 있습니다. 단, 레이아웃 설정은 기기별로 독립적으로 저장됩니다.

**Q: 조건부 주문은 서버에서 처리되나요, 제 컴퓨터에서 처리되나요?**
A: Bitget의 모든 주문(조건부 주문 포함)은 거래소 서버에서 처리됩니다. 따라서 컴퓨터를 종료하거나 브라우저를 닫아도 설정한 주문 조건이 유효하며, 조건이 충족되면 자동으로 실행됩니다.

**Q: API 키를 분실하거나 유출 의심이 들면 어떻게 해야 하나요?**
A: 즉시 'API 관리' 페이지로 이동하여 해당 API 키를 '삭제'하세요. 이는 가장 빠르고 효과적인 무효화 방법입니다. 이후 필요하다면 새 키를 생성할 수 있습니다. 평소에도 정기적인 키 순환을 보안 모범 사례로 고려해 보세요.
